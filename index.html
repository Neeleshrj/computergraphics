Problem Description

Creating a 3D Chess game, it's assets and making the game playable by learning Unity Game Engine, Blender for Asset creation and C# for the working of the game.


Video games and chess became sensations of every house after the COVID-19 pandemic and the Netflix series, Queen's Gambit. Inspired by these we would like to create our own version of Chess for our Computer Graphics and Multimedia Project Component.

Existing Approaches

Current approaches to creating chess include 2D Chess using a certain programming language,using a game engine and using javascript to create chess on browser.
